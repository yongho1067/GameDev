다운로드 / 실행 방법
Zips 폴더 -> 원하는 버전 -> 압축파일 다운 (분할파일 z0 ~ z# 전부 다운) -> 압축 해제 -> TileVania.exe 실행

Window 환경에서만 실행 가능
Mac, Andriod, IOS 실행 불가능
